# Welcome to Brautcloud Backend

Brautcloud Backend is a powerful, scalable service designed for wedding celebrations, allowing guests to easily share photos they capture during the event with the happy couple. Built with modern Java technologies, it provides a robust REST API for managing wedding events, users, and image uploads.

<Note>
  Brautcloud is currently in early development. Features and APIs may evolve as the project matures.
</Note>

## What is Brautcloud?

Brautcloud provides a backend infrastructure where wedding guests can:
- Upload photos they took at the wedding
- Share their captured moments with the bride and groom
- Access event-specific photo galleries
- Securely manage wedding event data

## Technology Stack

Brautcloud Backend is built with:

- **Java 25** - Latest Java features and performance improvements
- **Spring Boot 4.0.2** - Modern Spring framework for rapid development
- **PostgreSQL** - Robust relational database
- **AWS SDK for Java** - S3-compatible storage integration with AiStore
- **Maven** - Dependency management and build automation
- **Lombok** - Reduced boilerplate code
- **Docker Compose** - Containerized development environment
- **Doppler** - Secure environment variable management

## API Overview

The Brautcloud Backend exposes three main REST API endpoints:

### Events API

```
GET    /api/events          - List all events
POST   /api/events          - Create a new event
DELETE /api/events/{eventID} - Delete an event
```

### Images API

```
POST   /api/image           - Upload an image
GET    /api/image/{eventID} - Get all images for an event
DELETE /api/image/{imageID} - Delete an image
```

### Users API

```
GET    /api/users - List all users
POST   /api/users - Create a new user
```

## Project Status

Brautcloud Backend is actively developed and currently in early-stage development. The core features are functional, with ongoing improvements planned for:

- Enhanced security features (Spring Security integration)
- Advanced authentication mechanisms
- Photo processing and optimization
- Extended event management capabilities

<Warning>
  The Docker Compose Spring integration is currently disabled. You'll need to run `docker compose up` manually and ensure an AiStore container is running for S3-compatible storage.
</Warning>
