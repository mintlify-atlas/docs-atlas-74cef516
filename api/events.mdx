title: "Events API"
description: "API endpoints for managing wedding events in Brautcloud"
---

The Events API allows you to create, retrieve, and delete wedding events. Events are the core entity that organize images and connect to users.

## List All Events

<ParamField path="GET" type="endpoint">
  /api/events
</ParamField>

Retrieves a list of all events in the system.

### Response

Returns an array of event objects.

<ResponseField name="id" type="Long">
  Unique identifier for the event
</ResponseField>

<ResponseField name="eventName" type="String">
  Name of the event (e.g., wedding name)
</ResponseField>

<ResponseField name="location" type="String">
  Location where the event takes place
</ResponseField>

<ResponseField name="date" type="LocalDateTime">
  Date and time of the event in ISO 8601 format
</ResponseField>

<ResponseField name="userId" type="Long">
  ID of the user who owns this event
</ResponseField>

<ResponseField name="coupleName" type="String">
  Combined name of the couple (e.g., "John & Jane")
</ResponseField>

### Example Request

<CodeGroup>

```bash cURL
curl -X GET https://api.brautcloud.com/api/events \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.brautcloud.com/api/events', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});
const events = await response.json();
```

```python Python
import requests

response = requests.get('https://api.brautcloud.com/api/events')
events = response.json()
```

</CodeGroup>

## Create Event

<ParamField path="POST" type="endpoint">
  /api/events
</ParamField>

Creates a new wedding event.

### Request Body

<ParamField body="userId" type="Long" required>
  ID of the user creating the event
</ParamField>

<ParamField body="eventName" type="String" required>
  Name of the event
</ParamField>

<ParamField body="location" type="String" required>
  Location where the event takes place
</ParamField>

<ParamField body="date" type="LocalDateTime" required>
  Date and time of the event in ISO 8601 format (e.g., "2026-07-15T15:00:00")
</ParamField>

<ParamField body="password" type="String" required>
  Password to protect the event gallery
</ParamField>

<ParamField body="qrCode" type="String" optional>
  QR code data for event access
</ParamField>

### Response

Returns 200 OK on successful creation. No response body.

### Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.brautcloud.com/api/events \
  -H "Content-Type: application/json" \
  -d '{
    "userId": 5,
    "eventName": "Summer Wedding 2026",
    "location": "Garden Vista Hall",
    "date": "2026-07-15T15:00:00",
    "password": "secret123",
    "qrCode": "QR_CODE_DATA_HERE"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.brautcloud.com/api/events', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    userId: 5,
    eventName: 'Summer Wedding 2026',
    location: 'Garden Vista Hall',
    date: '2026-07-15T15:00:00',
    password: 'secret123',
    qrCode: 'QR_CODE_DATA_HERE'
  })
});
```

```python Python
import requests

data = {
    "userId": 5,
    "eventName": "Summer Wedding 2026",
    "location": "Garden Vista Hall",
    "date": "2026-07-15T15:00:00",
    "password": "secret123",
    "qrCode": "QR_CODE_DATA_HERE"
}

response = requests.post(
    'https://api.brautcloud.com/api/events',
    json=data
)
```

</CodeGroup>

## Delete Event

<ParamField path="DELETE" type="endpoint">
  /api/events/events/{eventID}
</ParamField>

Deletes an event by its ID.

<Note>
  Note: The actual endpoint path is `/api/events/events/{eventID}` due to the controller mapping. This will delete the event and all associated data.
</Note>

### Path Parameters

<ParamField path="eventID" type="String" required>
  The ID of the event to delete (will be parsed as Long)
</ParamField>

### Response

Returns 200 OK on successful deletion. No response body.

### Example Request

<CodeGroup>

```bash cURL
curl -X DELETE https://api.brautcloud.com/api/events/events/123 \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.brautcloud.com/api/events/events/123', {
  method: 'DELETE',
  headers: {
    'Content-Type': 'application/json'
  }
});
```

```python Python
import requests

response = requests.delete('https://api.brautcloud.com/api/events/events/123')
```

</CodeGroup>

## Data Models

### EventRequest

Used when creating a new event.

```json
{
  "userId": 5,
  "eventName": "Summer Wedding 2026",
  "location": "Garden Vista Hall",
  "date": "2026-07-15T15:00:00",
  "password": "secret123",
  "qrCode": "QR_CODE_DATA_HERE"
}
```

### EventResponse

Returned when retrieving events.

```json
{
  "id": 1,
  "eventName": "Summer Wedding 2026",
  "location": "Garden Vista Hall",
  "date": "2026-07-15T15:00:00",

```
